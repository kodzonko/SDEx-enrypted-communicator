# Jak uruchomić projekt

## Aplikacja kliencka

```shell
cd sdex-encrypted-communicator
npm install
npm start
```

## Aplikacja serwerowa

```shell
...
```

następnie z menu opcja:

- "w" - > wyświetla w aplikację w przeglądarce,
- "a" - > uruchamia na emulatorze Androida
