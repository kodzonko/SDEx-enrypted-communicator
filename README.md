# Instructions

1. Once download the project install the dependencies: `yarn install`

2. Second one install the pods: `cd ios/ && pod install`

3. Apply AndroidX packages: `npx jetify`

4. Run yarn: `yarn start`

5. Run on simulators:

  - iOS: `npx react-native run-ios`  - or - open the project directly from Xcode: `ios/Pulse.xcworkspace`

  - Android: `npx react-native run-android` - or - open the project directly from Android Studio and sync gradle
